
# <img src='./icons/LinkList.svg' height=25/> LinkList 
A google chrome extension for extracting links from websites.

## Features
- Extract all links from a website
- Extract links from selected text in a website
- Copies links to your clipboard(a link on each line)(examaple: you can use it with IDM, etc ...)  

## Installation
You can install this extension from chrome store.

## Usage
- Extracting all links in a webpage:
    - `Right click website > [LinkList] > Extract All Links from Page`  
    - From extension icon in the browser bar
- Extracting links from selected text
    - `Right click selected text > LinkList > Extract Links from Selection` 

## Development
1. **Clone this repo:**
    `git clone https://github.com/IPdotSetAF/LinkList`
2.  **Open Chrome Extensions:**
    Open Google Chrome and type `chrome://extensions` in the address bar, then press Enter.
3.  **Enable Developer Mode:**
    Toggle the switch for **Developer mode** in the top right corner.
4.  **Load the Extension:**
    Click the **Load unpacked** button (top left).
5.  **Select Folder:**
    Choose the folder `LinkList` where you cloned this repo.
